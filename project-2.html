NEON DISTRICT 7

ENVIRONMENT ART & LIGHTING

Project Overview

An atmospheric cyberpunk environment designed for real-time rendering in Unreal Engine 5. This project focuses on modular building systems, volumetric fog, and dynamic neon lighting to create a lived-in, dystopian city street.

Tools Used: Unreal Engine 5, Blender, Substance Designer
